# ROCK-PAPER-SCISSORS
This is a fun and interactive Rock Paper Scissors game created using HTML, CSS, and JavaScript. Players can compete against the computer with instant results, score updates, and smooth user interactions.
# Android Phone Server - Rock Paper Scissors

This project demonstrates hosting a Rock Paper Scissors web game using an Android phone as a local server through Termux and Python.

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Termux

## Features
- Local server hosting from Android phone
- Accessible through local Wi-Fi network
- Interactive game UI

## What I Learned
- Linux terminal basics
- Python HTTP server
- Networking concepts
- IP addressing
- Self-hosting basics
